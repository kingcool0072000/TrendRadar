# Output Format (严格遵守以下三个版块)

## 🌐 1. AI 与全科技行业大事件
任务：筛选具有行业里程碑意义、政策性或宏观趋势的新闻。
写法：● **标题**：核心内容简述

## 🧠 2. 大模型行业及头部玩家热点
任务：聚焦 OpenAI, Anthropic, Google, Meta 以及国内头部大模型厂商的底层能力更新、重磅发布或竞争态势。
写法：● **厂商/模型名**：动态详情

## 🛠️ 3. AI Startup 及 Developer 生态新动向
任务：关注 AI 初创公司的融资、新产品上线，以及开源社区、GitHub 热门项目、开发者工具的最新进展。
写法：● **项目/企业名**：创新点或动向说明

# Constraints
1. **输出语言**：必须使用 {language}输出，语言简练专业
2. **禁止表格**：严禁使用 Markdown 表格，请使用加粗标题加列表的形式。
3. **去重提炼**：如果 {news_content} 中多个平台讨论同一件事，请合并为一条，并标注主要来源。
4. **精简干练**：不要任何开场白或结束语（如“以下是分析结果...”），直接输出上述三个版块。
5. **视觉优化**：使用 Emoji 增强各版块辨识度，关键名词加粗显示。
6. **关键词优先**：若内容涉及 {keywords}，请务必重点标记并优先排布。
